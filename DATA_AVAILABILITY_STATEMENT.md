All simulation source code (Python 3 and NetLogo versions), parameter configuration files, and documentation required to reproduce the results reported in this study are publicly available in the following GitHub repository under the MIT License:

**Repository:** https://github.com/canslab1/BCAT

A persistent archived version with DOI is available via Zenodo:

**Zenodo DOI:** https://doi.org/10.5281/zenodo.REPLACE_WITH_ZENODO_DOI

The repository contains:
- `BCAT.py`: Python 3 implementation of the BCAT model with graphical user interface
- `English - best game no one played.nlogo`: Original NetLogo 4.0.5 implementation
- `test_scenarios/`: Parameter configuration files for reproducing all simulation scenarios presented in this paper (Figures 4, 5, 10, 11, and sensitivity analyses in Table 2 and Figures 7-9)
- `README.md`: Installation instructions, dependencies, and usage guide
- `requirements.txt`: Python package dependencies
- `CITATION.cff`: Machine-readable citation metadata

No human subjects data were collected in this study. All results are generated by computational simulation using the provided source code and parameter settings.

3. Link the GitHub repository to Zenodo (https://zenodo.org)
   - Log in to Zenodo with your GitHub account
   - Enable the BCAT repository in Zenodo settings
   - Create a new release on GitHub; Zenodo will automatically archive it and assign a DOI
4. Replace `REPLACE_WITH_USERNAME` and `REPLACE_WITH_ZENODO_DOI` in this template and in CITATION.cff
